# 量子纠缠交互实验室技术报告

## 1. 项目概述

量子纠缠交互实验室是一个用于量子纠缠实验、贝尔不等式检验与 CHSH 关联测量的交互式教学展示系统。作品将双光子纠缠实验从静态说明扩展为可旋转、可沉浸、可扫码、可调参的网页端展示平台，使观众能够从装置结构、光路传播、参数变化和计数结果四个层面理解实验。

系统的核心目标是帮助观众理解三个问题：纠缠光子如何产生，偏振分析如何影响符合计数，为什么 CHSH 的 S 值可以超过经典上界 `S <= 2`。

## 2. 系统组成

系统由五个前端展示模块和一个实验数据驱动模型组成：

- 3D 展示模式：自由观察量子纠缠实验装置和三维光路结构。
- VR 沉浸模式：以第一人称方式进入实验场景，观察纠缠光子产生、分析与探测过程。
- AR 扫描模式：通过 WebAR 图像识别，将动态实验装置叠加在真实环境中。
- 双光子纠缠 Demo：调节实验参数，实时观察单路计数、符合计数、关联函数与 CHSH 的 S 值变化。
- 3D Demo：通过 Three.js 三维交互演示补充理解实验光路与器件结构。
- 实验数据驱动的 CHSH 近似模型：为网页端计数变化、符合计数和 S 值提供连续预测。

## 3. 技术路线

项目采用网页端多模态展示路线：

- 使用 Blender 建立实验装置与光路相关三维资源。
- 使用 GLB / glTF 格式发布三维模型，便于在网页端加载。
- 使用 Three.js 进行三维渲染、相机控制、光效、粒子与路径动画展示。
- 使用 WebAR 图像识别实现移动端真实环境叠加。
- 使用 JavaScript、React 与 Vite 构建交互页面和 Demo 页面。
- 针对手机竖屏、横屏和桌面端分别进行布局适配。
- 使用 GitHub Pages 提供在线访问地址。
- 使用实验数据驱动模型实时生成教学演示中的计数与 CHSH 结果。

## 4. 物理模型

作品围绕双光子纠缠实验的主要物理过程展开：

- SPDC 自发参量下转换：405nm 泵浦光进入 BBO 晶体后产生相关光子对。
- BBO 晶体产生纠缠光子对：非线性晶体中的能量与动量守恒关系决定下转换光子的传播特征。
- 偏振分析：两路光子分别经过偏振分析器，偏振角设置会改变符合计数分布。
- 单光子探测：探测器记录两路单光子计数。
- 符合计数：时间窗口内同时探测到的光子对形成符合计数。
- CHSH 不等式：由四组角度下的关联函数计算 S 值。
- `S > 2` 的物理含义：当 S 值超过经典局域模型上界时，说明实验结果不能用经典局域隐变量模型解释。

## 5. 数据模型

网页端采用基于实测计数数据校准的 CHSH 教学近似模型。该模型以实验测得的单路计数、符合计数和 CHSH 角度组合数据为基础，结合偏振相关函数、探测效率、背景噪声和偶然符合项，生成适合交互展示的连续计数变化。

模型输入包括：

- 泵浦强度
- 偏振角
- 补偿状态
- 背景噪声
- 符合窗口
- 探测效率

模型输出包括：

- 单路计数
- 符合计数
- E 值
- S 值
- 可见度
- 偶然符合

该模型面向教学展示，用于表达实验参数变化对贝尔不等式检验结果的影响。它保留了物理规律上的可解释性，同时避免将网页演示误解为精密实验分析软件。

## 6. 交互设计

五个入口承担互补功能：

- 3D 展示模式强调空间结构和关键器件关系。
- VR 沉浸模式强调第一人称观察和实验过程体验。
- AR 扫描模式强调真实环境叠加和移动端现场展示。
- 双光子纠缠 Demo 强调参数调节、计数变化和 CHSH 结果。
- 3D Demo 强调实验光路与器件结构的补充理解。

这种设计使观众既能看到实验装置，也能操作参数并观察结果变化，从而形成从“看见光路”到“理解计数”再到“理解 S 值”的认知路径。

## 7. 移动端展示

移动端展示重点面向扫码体验。首页提供清晰入口，AR 页面提供识别图、摄像头授权、扫描状态和异常提示。双光子纠缠 Demo 对光路示意图和控制区域进行适配，使手机竖屏下仍能观察主要光路结构并上下滑动查看参数和说明。

## 8. 创新点

- 将微视频叙事、三维建模、WebVR、WebAR 与实验数据驱动模型结合为同一展示系统。
- 展示方式从静态观看扩展为可交互、可扫码、可参数调节的教学平台。
- 双光子纠缠 Demo 使用实测数据校准的计数与 CHSH 近似模型，使网页端结果更贴近真实实验规律。
- 五个展示入口覆盖桌面端、移动端和现场扫码场景，适合教学和竞赛展示。
- 系统把抽象的贝尔不等式检验结果转化为可观察的计数变化和可解释的 S 值变化。

## 9. 局限与改进

当前数据模型为教学近似模型，主要用于展示参数变化趋势和 CHSH 结果关系。未来可以接入更多实测角度数据，进一步完善误差分析、统计不确定度和探测器参数标定。后续也可以接入真实实验采集系统，使网页展示与实时实验数据联动。
