# 实验数据驱动的 CHSH 近似模型说明

## 1. 数据来源

模型以实验测量得到的单路计数、符合计数和 CHSH 角度组合数据为基础。数据用于校准网页端展示中的计数规模、角度相关性、可见度变化和 S 值变化，使交互结果与双光子纠缠实验规律保持一致。

## 2. 模型目的

该模型用于教学演示和交互展示，帮助观众理解实验参数如何影响单路计数、符合计数、关联函数和 CHSH 的 S 值。它不替代精密实验分析，也不作为实验误差评估的最终工具。

## 3. 模型形式

模型由四类因素共同构成：

- 偏振相关函数：描述两路偏振角差异对关联函数和符合计数分布的影响。
- 探测效率：描述探测器对单路计数和符合计数规模的影响。
- 背景噪声：描述非理想环境下随机计数的引入。
- 偶然符合：描述符合窗口内非同源光子造成的额外符合计数。

通过这些因素，网页能够在滑块连续变化时生成平滑且可解释的计数结果。

## 4. CHSH 计算方式

CHSH 检验使用四组偏振角组合计算关联函数：

- `E(a, b)`
- `E(a, b')`
- `E(a', b)`
- `E(a', b')`

S 值由四项关联函数组合得到：

```text
S = |E(a,b) - E(a,b') + E(a',b) + E(a',b')|
```

当 `S > 2` 时，结果超过经典局域模型给出的上界，体现量子纠缠关联的非经典特征。

## 5. 滑块如何影响计数和 S 值

- 泵浦强度提高时，单路计数和符合计数通常上升。
- 偏振角变化会改变关联函数 E，从而改变 CHSH 的 S 值。
- 补偿状态影响可见度，补偿不足会降低关联清晰度。
- 背景噪声增加会抬高随机计数并削弱有效关联。
- 符合窗口增大会提高计数规模，同时可能增加偶然符合。
- 探测效率影响两路探测器记录到的计数比例。

## 6. 模型可信度说明

模型以实测计数数据校准，并在形式上保留偏振相关、探测效率、背景噪声和偶然符合等实验因素，因此适合展示参数变化趋势与 CHSH 结果关系。它的可信度体现在物理趋势一致、输出连续可解释、与教学目标匹配。

需要注意的是，该模型是教学近似模型，不等同于完整实验数据处理流程。若用于科研级分析，还需要更完整的误差传播、系统误差标定和实验重复测量。

## 7. 网页中的使用方式

双光子纠缠 Demo 和 3D Demo 共享同一套模型口径。页面根据用户调节的泵浦强度、偏振角、补偿状态、背景噪声、符合窗口和对准程度实时计算输出，展示单路计数、符合计数、E 值、S 值、可见度和偶然符合。

这种方式使观众能够直观看到参数变化如何影响贝尔不等式检验结果。
