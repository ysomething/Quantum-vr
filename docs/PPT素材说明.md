# PPT 素材与表述建议

## 推荐标题

量子纠缠交互实验室：面向贝尔不等式检验的 3D/VR/AR 教学展示系统

## 技术栈表述

- Blender：实验装置建模与动画
- GLB / glTF：三维资源发布
- Three.js：网页端三维渲染
- WebAR：图像识别与现实叠加展示
- JavaScript / React / Vite：交互网站开发
- 实验数据驱动模型：计数与 CHSH 结果预测

## 创新点表述

- 从微视频拓展到可交互实验环境。
- 从静态科普拓展到参数可调的教学模型。
- 从二维展示拓展到 3D、VR、AR 多模态体验。
- 由实测数据校准网页计数结果，提高演示可信度。
- 将抽象的贝尔不等式检验转化为可观察、可操作、可讲解的网页交互体验。

## 可放入 PPT 的一句话

系统将量子纠缠实验从视频叙事拓展为可交互、可扫码、可参数调节的多模态教学展示平台。

## 答辩口径建议

可以从“物理实验教学难点”切入：量子纠缠实验装置复杂，贝尔不等式和 CHSH S 值抽象，单纯视频难以体现参数与结果的关系。本系统通过三维空间展示、沉浸式观察、AR 叠加和数据驱动 Demo，把实验结构和计数变化联系起来，使观众能够边看边操作、边观察边理解。
