# 展示说明与答辩要点

## 1. 一句话介绍

本作品构建了一个面向量子纠缠实验教学的交互式展示系统，融合三维建模、VR/AR 展示与实测数据驱动的 CHSH 计数预测模型。

## 2. 推荐展示流程

1. 打开首页，说明系统包含五个入口。
2. 进入 3D 展示模式，讲解实验装置空间结构。
3. 进入 VR 沉浸模式，展示第一人称观察实验光路。
4. 进入 AR 扫描模式，用手机对准识别图观察现实叠加效果。
5. 进入双光子纠缠 Demo，调节滑块并展示计数与 S 值变化。
6. 说明数据模型来自实测计数数据校准。
7. 回到物理意义：`S > 2` 表示实验结果超过经典局域模型上界。

## 3. 各模式讲解要点

### 3D 展示模式

- 该模式用于观察实验装置整体结构。
- 重点说明 405nm 泵浦光进入 BBO 晶体后产生纠缠光子对。
- 通过旋转视角观察偏振分析器、探测器和符合计数模块。
- 自动导览帮助观众按实验流程理解光路。

### VR 沉浸模式

- 该模式强调第一人称视角和沉浸式观察。
- 观众可以从实验场景中观察纠缠光子路径。
- 适合说明实验过程不是单个元件，而是一整套空间光路系统。
- 手机端可使用普通沉浸或体感方式体验。

### AR 扫描模式

- 该模式用于现场扫码展示。
- 识别图相当于实验装置的现实锚点。
- 识别成功后，动态装置叠加到真实环境中，便于移动端传播和现场讲解。
- 该模式体现了实验教学与 WebAR 展示技术的结合。

### 双光子纠缠 Demo

- 该模式用于解释参数变化与计数结果的关系。
- 调节偏振角可以观察关联函数和 CHSH 的 S 值变化。
- 调节补偿、噪声和符合窗口可以观察可见度和偶然符合变化。
- 计数与 S 值由实测数据校准的 CHSH 近似模型实时生成。

### 3D Demo

- 该模式用三维交互补充说明光路和器件位置。
- 页面中 D1、D2、符合计数和 CHSH S 值会随参数变化。
- 它与双光子纠缠 Demo 共享同一套数据驱动模型口径。

## 4. 可能问题与回答

### 本网站和视频是什么关系？

网站把量子纠缠实验从线性视频讲解扩展为可交互展示。视频适合叙事，网站适合观察、操作和现场演示。

### 数据是怎么来的？

网页端计数和 S 值由实测计数数据校准的 CHSH 教学近似模型实时生成。模型用于教学展示，表达参数变化与计数结果之间的物理关系。

### 为什么 S 会超过 2？

CHSH 不等式中，经典局域模型给出上界 `S <= 2`。纠缠态在合适的偏振角组合下可以得到 `S > 2`，说明实验关联不能由经典局域隐变量模型解释。

### AR 和 VR 的区别是什么？

VR 沉浸模式是在虚拟三维场景中观察实验；AR 扫描模式是在真实摄像头画面上叠加动态实验装置。

### 该模型是否等同于真实实验分析？

该模型面向教学演示，基于实测数据校准并保留物理规律趋势，但不替代精密实验中的完整误差分析和仪器标定。

### 创新点在哪里？

作品将三维建模、沉浸式展示、WebAR 图像识别与实验数据驱动模型组合在同一网页系统中，使量子纠缠实验可以被观察、扫码、操作和解释。
